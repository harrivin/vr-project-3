Description and Motivation

The project presented is a virtual reality visualization of the dataset on the per capita income(all races) in The United States from the year 1967-2014. The project begins with a scene that contains a house and a garage, it also describes how the tangible assets in the house and the garage change from year to year based on the per capita income. The motivation behind the project is to visualize a dataset and predict how the lifestyle of an average person living The United States changes from year to year based on the per capita income. The given experience uses first person perspective, head tracking and stereoscopic rendering to put the user in the Visualization scene. The user can move around the scene using a X-box Joystick(left stick) and can change the value of the slider(years) using "A" and "X" buttons.

Dataset Used

The Dataset used is from the Current Population Survey and contains the Per Capita Income of All Races from 1967 - 2014 Link :- http://www.census.gov/hhes/www/income/data/historical/people/2014/p01AR.xls
